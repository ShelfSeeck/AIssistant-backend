# Labs Guide

Create your lab files in this folder:

- `lab_stage1_core.py` ... `lab_stage8_core.py` (required core labs)
- `lab_stageX_applied.py` (optional applied labs)

Suggested workflow:
1. Read one stage note.
2. Implement core lab in this folder.
3. Run the lab file and capture output logs.
4. Ask for mentor review with your code and output.
